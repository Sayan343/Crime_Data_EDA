# Crime Data Analysis


### Data Sources
This project is based on performing and presenting exploratory data analysis (eda) based on a dataset from a government source.

### Overview
To inspect the dataset for missing values, outliers, inconsistencies, and
underlying trends. Through visualizations and summary statistics.We aim to uncover meaningful insights, identify relationships between key variables, and understand the overall behavior of the data.

### Tools
- Python Libraries
- Pandas - Data cleaning
- Numpy - Handling Missing values & Statistical Anlysis
- Matplotlib & Seaborn - Visualization
- Word Cloud - Visual Represtation
